# Privacy Policy for Proxy Manager
This extension is designed to provide a simple interface for managing HTTP proxies.

### Data Collection & Usage
- **Personal Data**: This extension **does not collect**, store, or transmit any personal information, browsing history, or user data to external servers.
- **Proxy Credentials**: All proxy details (IP, port, username, password) are stored locally on your device using `chrome.storage.local`.
- **Authentication**: Credentials are used solely for proxy authentication via the official Chrome `webRequest` API.

### Security
- No tracking scripts or third-party analytics are included.
- Your data stays on your machine.
